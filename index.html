What does it sound like to a screen reader?

The exact words are lifted from VoiceOver on Mac OS, but are good for a general guide.

Here are some examples of a label and an input in various states, and what they sound like to a screen reader user.
What it is What it looks like What it sounds like
input wrapped in label Username, edit text.
No for and id attributes edit text.
With for and id attributes Username, edit text.
Required with aria-required="true" Username, required, edit text.
Required with aria-required="true", error state using aria-invalid Username, required, invalid data, edit text.
Error message in label Username, Please enter your username, edit text, invalid data.
Error message before input, linked with aria-describedby Please enter your username. Username, invalid data, edit text. (pause) Please enter your username.
Error message after input, linked with aria-describedby Please enter your username. Username, invalid data, edit text. (pause) Please enter your username.
